# Cut Spark React App

This is a React transformation of the Cut Spark HTML website.

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

The app will open at http://localhost:3000

## Building for Production

```bash
npm build
```

## Project Structure

```
src/
  components/
    - Navigation.jsx
    - Home.jsx
    - About.jsx
    - Work.jsx
    - Services.jsx
    - ProjectRegistration.jsx
    - Blog.jsx
    - Contact.jsx
    - Footer.jsx
  App.js
  index.js
  index.css
```

## Notes

- All original CSS, images, and videos are preserved in their original locations
- The original JavaScript plugins need to be integrated or replaced with React equivalents for full functionality
- Forms are set up with React state management
